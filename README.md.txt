# AI-Powered Marketplace Assistant for Local Artisans

## Overview
This prototype helps local artisans market their crafts with AI-powered descriptions and a simple web interface.

## Tech Stack
- Flask (Backend)
- Google Gemini AI (AI Integration)
- HTML Templates (Frontend)
- Google Cloud Run (Deployment)

## Features
- Artisan can input product details.
- Gemini AI generates attractive descriptions.
- Outputs displayed in browser.

## Run Locally
```bash
pip install -r requirements.txt
python app.py
